MChess

A website of chess experiments

What is MChess?

Work on MChess began on the 10th anniversary of Stockfish. The main goal is to develop my 'TensorChess' engine, to explore work on zero-knowledge machine learning.

This project also allows me to explore work on CSS, JS, Flask and many other things I never really got the chance to play with before. This is a learning exercise for me, and if sucessful may blossom into a rather fun chess website.

What is TensorChess?

AlphaZero showed the world that chess can be learned by a machine with zero knowledge of the game.

TensorChess will be an ongoing project to make a zero-knowledge engine that can be run on inexpensive hardware.
